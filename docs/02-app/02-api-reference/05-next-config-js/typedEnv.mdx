nav_title: typedEnv
description: Enable type-checking and autocompletion for environment variables.
---

Enable type-checking and autocompletion for environment variables. You can opt out by setting the `typedEnv` value to false. This feature requires using TypeScript in your project.

```js filename="next.config.js"
/** @type {import('next').NextConfig} */
const nextConfig = {
  // enabled by default, opt out by setting to false
  typedEnv: false,
}

module.exports = nextConfig
```
